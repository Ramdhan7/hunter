# hunter
Skrip dengan clone super cepat dan tampilan yang lebih keren



### Install :
````bash
$ pkg update && pkg upgrade 
$ pkg install python2 
$ pkg install git 
$ pkg install bs4
$ pip2 install mechanize 
$ pip2 install requests 
$ git clone https://github.com/Ramdhan7/hunter
````
### Run Script :
````bash
$ cd hunter
$ ls
$ python2 hunter.py

````
